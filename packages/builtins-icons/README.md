# `@aeriajs/builtins-icons`

## Introduction

The code of this package is generated with the `compile-builtins-icons.ts` script.
This package exports icons used by `@aeriajs/builtins`.

## Usage

```typescript
import { icons } from '@aeriajs/builtins-icons'
// ['user', 'trash', ...]
console.log(icons)
```
